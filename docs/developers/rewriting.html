Rewriting code

Rewriting source code with Scalafix can have a difficult learning curve, but once you get the hang of it, it can be fun and rewarding. This document should help you learn the steps to become effective rewriting source code with Scalafix.

The steps to rewrite source code roughly fall into

  1. write a minimal test case
  2. find the target tree nodes to rewrite
  3. construct a Patch to describe rewrite

Producing a diff with Patch

Producing a diff with Patch

Rewrites are described using a data structure called Patch, which is

  • immutable: it's a sealed data structure similar to Option and Either.
  • composable: you can combine two patches with patch1 + patch2.
